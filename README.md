# Notas:
Aplicacion de consola interactivo para manejo de tareas
```
============================
   Seleccione una opcion
============================
? Que desea hacer? (Use arrow keys)
> 1. Crear tarea
  2. Listar tareas
  3. Listar tareas completadas
  4. Listar tareas pendientes
  5. Completar tarea(s)
  6. Borrar tarea
  0. Salir
(Move up and down to reveal more choices)
```
